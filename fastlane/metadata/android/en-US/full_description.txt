Ollama Android runs large language models (LLMs) directly on your device — completely offline, no cloud, no data leaves your phone.

Powered by llama.cpp, the app delivers fast on-device AI inference optimized for ARM processors with KleidiAI acceleration, OpenMP multi-threading, and NEON SIMD instructions.

Features:
- Run Gemma 3, Qwen 2.5, SmolLM2, and other GGUF models locally
- Streaming chat interface with real-time token generation
- Download and manage models directly from Hugging Face
- Optimized for Snapdragon and other ARM64 processors
- KleidiAI-accelerated quantized matrix operations
- Battery optimization setup wizard for reliable background inference
- Foreground service keeps inference alive
- No internet required after model download
- No telemetry, no tracking, no cloud dependency

Recommended devices:
- 6GB+ RAM for 1B parameter models (Q4_K_M quantization)
- 8GB+ RAM for 4B parameter models
- ARM64 (arm64-v8a) processor required

This is a fully open-source project. All inference happens on-device using your CPU — your conversations never leave your phone.
